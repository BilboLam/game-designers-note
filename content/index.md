# Game Designers&#x27; Note
**Game Designers&#x27; Note** is a personal knowledge base for game design, inspired by [The Level Design Bookarrow-up-right](https://book.leveldesignbook.com/), and aims to compile my scribbled game design notes into a structured collection. These are simply my personal views, shared as a starting point for thinking, not as definitive answers.

circle-infoMany notes were taken from videos and other sources over time, and some references have been lost along the way. If you recognize any uncredited content, feel free to reach out at bilbo_bagginss_@outlook.com.
