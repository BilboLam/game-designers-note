# Map System
#### [hashtag](#control-map-elements)Control Map Elements
- Auto-annotate as players explore

Show rooms visited, doors locked, items found.

- Remove outdated annotation

Clean up annotations when an area is clear so players know it&#x27;s empty. 

- Show more POI as the game progresses

Don&#x27;t show everything at once.
