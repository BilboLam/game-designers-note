# Platformer
#### [hashtag](#celeste-game-feel-tricks)Celeste Game Feel Tricks
- Coyote time

- Jump buffering

- Halved gravity at jump peak — allows adjusting for landing

- Jump corner correction / dash corner correction

- Lift momentum storage — store momentum for a few frames after platform stops

- Wall jump: 2px from wall; super wall jump: 5px (in 320×180 game)
